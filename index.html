Early Access Installation Guide

While FlashDeal is being reviewed by the Chrome Web Store, you can install our beta version using developer mode:

  1. Download the extension ZIP file: Download FlashDeal
  2. Unzip the file to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" using the toggle in the top-right corner
  5. Click "Load unpacked" and select the folder containing the unzipped extension files
  6. FlashDeal is now installed and ready to save you money!

Note: Since this is a developer mode installation, you may see a reminder about using developer mode extensions when you start Chrome. This is normal and you can dismiss this notification.
